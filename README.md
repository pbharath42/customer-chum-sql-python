customer-churn-analysis-sql-python
Customer Churn Analysis using SQL and Python

Overview
This project analyzes customer churn data to identify patterns that lead to customer attrition.

 Tools Used
- SQL (data extraction & analysis)
- Python (Pandas, Matplotlib)
- Jupyter Notebook

 Project Workflow
1. Created customer churn dataset
2. Performed churn analysis using SQL queries
3. Imported SQL data into Python
4. Analyzed churn distribution
5. Visualized churn using bar charts

 Key Insights
- Majority of customers did not churn
- Significant churn observed in a specific segment
- Churn pattern clearly visible through visualization

 Files
- customer_churn.sql – SQL queries
- customer_churn_analysis_python.ipynb – Python analysis
- Screenshots – SQL & Python outputs

 Conclusion
This project demonstrates practical skills in SQL querying, Python analysis, and data visualization.
